# YoloAnnotator
可以实现图片数据的手动打标，然后使用打标好的数据训练yolo模型，再使用训练好的模型进行推理量化。

项目的.pro文件引用到了OpenCV-MinGW-Build-OpenCV-4.5.5-x64和onnxruntime-win-x64-1.25.1，可以在github上下载，然后配置好路径。
